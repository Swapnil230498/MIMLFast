This package includes the PYTHON codes of MIMLfast, which is a fast algorithm for multi-instance multi-label learning.

I have included a sample data and a kaggle data in the package. To get start with the codes, please open 'Main Code file'. 

Reference, For the detail:
S.-J. Huang W. Gao and Z.-H. Zhou. Fast multi-instance multi-label learning. In: Proceedings of the 28th AAAI Conference on Artificial Intelligence (AAAI'14), 2014.
